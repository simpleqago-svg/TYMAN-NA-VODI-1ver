## 2. Шапка (header)

**Элементы:**

| Элемент | Действие |
|---------|----------|
| Логотип / название | Tyman na Vodi |
| Меню | → страница «Меню» |
| Instagram | → профиль заведения |
| Кнопка | **Забронировать стол** → resOS |

**Sticky-кнопка:** «Забронировать стол» — видна при прокрутке главной страницы (особенно на мобилке).

## 3. Блоки главной страницы

### Блок 1 · Hero

**Назначение:** первое эмоциональное впечатление + главное действие.

| Элемент | Содержание |
|---------|------------|
| Фон | Атмосферное видео или сильное фото с гостями / вечерним светом |
| Заголовок | Tyman na Vodi |
| Подзаголовок | 1 строка позиционирования |
| CTA | **Забронировать стол** (единственная кнопка) |

**Не на Hero:** «Посмотреть меню», «Маршрут», «Узнать больше».

---

### Блок 2 · Почему сюда

**Назначение:** быстро объяснить, чем место отличается от обычной кальянной.

**4 карточки:**

1. **Атмосфера для вечера** — пространство, где можно спокойно провести время, пообщаться и остаться дольше.
2. **Кальяны как часть отдыха** — часть общего впечатления от вечера, не просто услуга.
3. **Кухня и напитки в одном месте** — всё для вечера уже внутри, не нужно искать отдельный бар или ресторан.
4. **VIP-комната для компании** — отдельное пространство для своего круга.

**CTA после блока:** Забронировать стол.

---

### Блок 3 · Атмосфера + сценарии

**Назначение:** показать место визуально и дать гостю узнать «свой» сценарий.

**Галерея:** 6–10 фото — гости, свет, кальяны, еда, напитки, детали сервировки.

**4 сценария** (можно встроить в визуал):

| Сценарий | Текст |
|----------|-------|
| Встреча с друзьями | Кальяны, напитки, кухня — весь вечер в одном месте |
| Свидание | Спокойный свет, музыка, можно поговорить |
| День рождения | Стол, кальяны, напитки, бронь заранее |
| День без спешки | Днём — кофе и работа, вечером — остаться |

---

### Блок 4 · VIP-комната

**Назначение:** показать VIP как отдельный продукт, а не «ещё одну зону».

| Элемент | Содержание |
|---------|------------|
| Фото | Комната, большой экран, PS5, посадка |
| Заголовок | VIP-комната для вашей компании |
| Описание | Изолированное пространство для компании |
| Список | Отдельная комната · большой экран · PS5 · актуальные игры · трансляции · день рождения · бронь по депозиту |
| CTA | **Забронировать VIP-комнату** → resOS |

---

### Блок 5 · Меню (краткий обзор)

**Назначение:** показать, что заведение подходит для полноценного отдыха.

- Категории: **Кальяны · Кухня · Напитки**
- CTA: **Посмотреть меню** → страница «Меню»

Полное меню на главной **не раскрываем**.

---

### Блок 6 · Отзывы

**Назначение:** доверие, снятие сомнений.

**Формат:** 7–10 карточек, слайдер или сетка.

**Каждая карточка:**

- ⭐ Рейтинг (звёзды)
- Цитата (2–4 строки)
- Фото гостя или аватар с инициалами
- Имя
- Источник (Google)
- Дата (опционально)

**Реализация:** блок «Отзывы» в Tilda, контент вручную из лучших отзывов Google.

---

### Блок 7 · Кейтеринг / корпоративы

**Назначение:** информировать о формате и собирать лиды.

| Элемент | Содержание |
|---------|------------|

**Схема:**

```
[Форма] → заявка заведению
       → «Спасибо» → [Скачать PDF условий]
```

## 4. Страница «Меню»

**Назначение:** для гостей, которые хотят изучить меню до визита.

**На старте:** PDF меню (кнопка открыть / скачать).

**Позже (опционально):** структурированная страница:

- Кальяны
- Кухня
- Напитки
- Десерты

**Требования:** простота, читаемость, без перегрузки.

## 5. Интеграции

| Система | Где используется | Как |
|---------|------------------|-----|
| **resOS** | Hero, sticky, шапка, VIP, триггер | Ссылка или виджет + UTM |
| **Tilda Forms** | Кейтеринг / корпоративы | Заявка + success page |
| **Google Maps** | Footer | Embed карты |

### resOS — детали

- Кнопки «Забронировать стол» → resOS (`utm_source=website&utm_campaign=book_table`)
- Кнопка VIP → resOS (`utm_campaign=book_vip`)
- Sticky-кнопка на всём лендинге
- Тест: бронь доходит в систему

### Кейтеринг — детали

- Поля формы: имя, телефон
- Получатель заявки: email / Telegram / WhatsApp (настраивается)
- Success page: кнопка скачивания PDF условий
